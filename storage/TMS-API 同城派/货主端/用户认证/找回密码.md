```
{{url}}/merchant/password/reset
```
### 请求方式: PUT

 |---|---|---|---|  
|email|""|是|"whoyummy2@aliyun.com"|  
|code|""|是|"849580"|  
|new_password|""|是|"12345678"|  
|confirm_new_password|""|是|"12345678"|
