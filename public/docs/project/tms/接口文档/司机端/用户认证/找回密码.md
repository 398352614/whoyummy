```
{{url}}/driver/password-reset
```
### 请求方式: PUT  
### 请求参数:  

|参数名|说明|必填|示例|  
 |---|---|---|---|  
|email|""|是|"398352614@qq.com"|  
|code|""|是|"315330"|  
|new_password|""|是|"12345678"|  
|confirm_new_password|""|是|"12345678"|  
|_method|""|是|"PUT"|
